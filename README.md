# NEFARAM Local Mods

Source and packaged files for local NEFARAM add-on mods.

Source of record: use this repository at `C:\Users\antho\nefaram-files` for source edits, commits, tags, and GitHub releases. Treat `C:\games\nefaram` as an installed MO2 runtime/reference tree, not the release source.

| Mod | Folder | Purpose | Source | Build | Install |
| --- | --- | --- | --- | --- | --- |
| RaceMenu Appearance Slots | `RaceMenu Appearance Slots` | Stores two player-authored RaceMenu appearance slots and toggles between them. | `Source/Scripts` plus `RMAppSlots.esp`. | Compile the Papyrus sources against RaceMenu, SkyUI, PapyrusUtil, and PO3 Papyrus Extender sources; current `.pex` files are included. | Copy the folder to MO2 `mods`, enable it, and enable `RMAppSlots.esp`. |
| RaceMenu Appearance Slots - Vampire Lord Bridge | `RaceMenu Appearance Slots - Vampire Lord Bridge` | Applies Appearance Slots slot 2 on Vampire Lord transform and slot 1 on revert. | `Source/Scripts/HNVVampLord.psc`. | Compile the loose-script override against Humanoid Vampire Lords and RaceMenu Appearance Slots sources. | Copy to MO2 `mods` and place after Humanoid Vampire Lords so `Scripts/HNVVampLord.pex` wins. |
| Spatial Storage Rings | `Spatial Storage Rings Work` | Adds storage rings that open one shared container with tiered item limits. | `Spatial Storage Rings/Source/Scripts`, `SSEEdit Scripts`, and `ssr-mutagen-build`. | Run `Spatial Storage Rings/Build-CompilePapyrus.ps1` for Papyrus; rebuild the ESP with the Mutagen project or SSEEdit script if plugin records change. | Copy `Spatial Storage Rings Work/Spatial Storage Rings` to MO2 `mods` and enable `Spatial Storage Rings.esp`. |
| SexLab Humanoid Vampire Lord Female Player Patch | `SexLab Humanoid Vampire Lord Female Player Patch` | Makes the female player in Vampire Lord race classify as a humanoid female in SexLab. | `Scripts/Source/sslActorLibrary.psc`. | Compile the patched `sslActorLibrary.psc` against SexLab Framework sources. | Copy to MO2 `mods` and place after SexLab Framework so `Scripts/sslActorLibrary.pex` wins. |
| SPERG Script Lag Patch | `SPERG-Script-Lag-Patch` | Reduces Papyrus backlog from SPERG weapon-speed updates triggered by unrelated magic effects. | `SPEWeaponSpeedScript.psc` and `SPEWeaponSpeedScript-OnMagicEffectApply.patch`. | Compile `SPEWeaponSpeedScript.psc` against SPERG and vanilla sources. | Copy to MO2 `mods` and place after SPERG so the loose `scripts/SPEWeaponSpeedScript.pex` overrides the BSA copy. |
| Lola Expanded Addons | `lola-expanded-addons` | Merged pluginless Lola integrations for forced tasks, body potion routines, fertility, milk economy, and configuration. | `Source/Scripts`, `SKSE/Plugins/LolaExpandedAddons`, and `tools/generate-hair-pool.mjs`. | Compile the Papyrus overrides against Submissive Lola, Custom Framework Lola Addon, PapyrusUtil, and integration mod sources; rerun the hair-pool generator when changing hair data. | Copy to MO2 `mods`, enable it, and keep the older separate Lola addon mods disabled. |
| Bodymorph Alterations | `dollform` | Adds self-cast Alteration bodymorph transformation powers, progression, SlaveTats overlays, and MCM repair tools. | `Source/Scripts`, `Dollform.esp`, SlaveTats metadata/textures, and notes. | Compile Papyrus sources against SkyUI, RaceMenu/NiOverride, PapyrusUtil, PO3 Papyrus Extender, SlaveTats, and vanilla sources; edit plugin records in xEdit/CK if forms change. | Copy to MO2 as `Bodymorph Alterations`, enable `Dollform.esp`, and run `bat BodymorphStartMCM` once on existing saves if the MCM is missing. |
| MO2 Mod Group Installer | `tools/ModGroupInstaller` | Windows tool for installing shared mod groups into a portable MO2 instance from a simple manifest. | C#/.NET WinForms app plus example manifests in `examples`. | Run `dotnet build tools/ModGroupInstaller/ModGroupInstaller.csproj -c Release`. | Run the built exe, choose the MO2 root and manifest, or use CLI `--mo2` and `--manifest`. |
| MO2 Mod List Compare | `tools/ModListCompare` | Compares another MO2 `modlist.txt` against the local NEFARAM install and drafts installer manifests for missing mods. | Python script plus generated comparison output in `out`. | Run `python tools/ModListCompare/compare_modlists.py --source-modlist C:\Users\antho\Downloads\modlist.txt --mo2-root C:\Games\nefaram`. | Review `out/missing.md`, then feed selected `out/manifests/*.mods.txt` files to the Mod Group Installer. |

Each listed folder has a README with purpose, source, build, and install notes.
